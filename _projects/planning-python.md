Planning is an open-source Python path-planning library with a unified architecture for sampling-based planners and 3D visualization.

- Built a consistent planner interface across RRT-family and PRM-family algorithms.
- Implemented informed sampling, triangle-inequality heuristics, and diffusion-based trajectory planning examples.
- Added visualization support to compare planner behavior across environments.
