Hongdo is an Ansan mascot promotional robot designed for public-space human-robot interaction at outdoor events.

- Contributed to the ROS software stack for a mobile mascot robot that roams public spaces and approaches detected people.
- Integrated robot behavior with an event-facing UI so visitors could interact with the robot and take photos.
- Connected web, vision, and AI image-transformation components for on-site interactive experiences.
- Supported a multi-system deployment spanning NUC-side ROS/web modules and Jetson-side vision processing.
